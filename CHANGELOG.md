# 更新日志

## v1.0.3

- 首页调整为店铺优先布局，按店铺、报表类型和年月区间查询汇总与明细。
- 每个店铺使用独立 SQLite 数据库，支持数据隔离和大数据查询优化。
- 店铺名称支持直接编辑，历史数据和配置会同步更新。
- 参数配置支持维护原始字段、汇总字段、冻结栏/活动栏、计算口径和分页行数。
- 支持 Excel/CSV 导入，导入时统一选择店铺、报表类型和归属年月。
- CSV 导入兼容 UTF-8 BOM 和 GB18030，导入时清理多余前导文本标记。
- 明细流水支持分页、跳页、表头排序、多字段精确筛选和筛选条件暂存后统一执行。
- 汇总表支持冻结区与活动区拖动调宽，列宽按内容适配。
- 差异明细改为表格化页面，并保留差异原因、调整字段和调整说明。
- 导出改为分批读取并流式写入 XLSX，导出进度在状态栏显示。
- 导出文件和汇总 sheet 顶部包含店铺、报表类型、年月区间和导出时间。
- 导出原始表格空值保持为空，订单编号等字段去掉多余前导单引号。
- 全局设置支持开机自启动和点击 X 最小化到托盘。
- 新增备份还原功能，可备份/还原全部或部分店铺数据库及配置。
- 弹窗按钮改为中文业务文案，例如“开始导入”“保存余额”“退出”。
- README 新增使用动画，便于快速了解程序流程。
- 明细筛选条件支持换行显示，避免筛选条件过多时被挤在一行看不全。
- 明细流水表格与底部筛选/分页/状态区域之间支持拖动调整高度。
- 明细分页按钮与编辑/删除流水操作合并到同一行。
- 导入完成后状态栏显示导入完成、新增行数和跳过行数。
- 明细按钮文案调整为“加入添加筛选条件”“清除筛选条件”。

## v1.0.2

- 修复部分抖音店铺 Excel 因字段别名不同导致导入失败的问题。
- 导入时支持更多动账字段、摘要字段、订单字段和服务费字段别名。
- 自动补充店铺原始字段配置，后续可在参数配置中调整。

## v1.0.1

- 汇总表新增调整说明。
- 导出升级为 XLSX，并包含汇总表和原始表格。
- 支持多年月区间查询。
- 区间汇总中期初取首月期初，期末取末月期末，不再累加。
- 支持按店铺维护原始字段、汇总字段和计算口径。

## v1.0.0

- 首个 Qt/PySide6 Windows 桌面版本。
- 支持店铺配置、Excel 流水导入、SQLite 本地落盘。
- 支持按店铺和月份汇总、余额核对、差异明细、手工调整和历史查询。
