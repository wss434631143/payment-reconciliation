# 更新日志

## v1.0.2

- 修复抖音店铺 Excel 导入时因 `动帐时间1`、`动帐流水号`、缺少 `动账摘要` 导致失败的问题。
- 导入时支持更多字段别名自动识别：`动账/动帐`、`动账场景`、`计费类型`、`订单号/子订单号`、`平台服务费` 等。
- 抖音表格中的原始字段会自动写入店铺字段配置，后续可在店铺配置中调整。

## v1.0.1

- 汇总表新增“调整说明”，显示手工调整记录。
- 导出从 CSV 升级为 XLSX，并包含“汇总表”和“原始表格”两个 sheet。
- 月份筛选支持多选和区间输入，例如 `202606-202607`。
- 多月份查询时按店铺显示区间合计，并保留单月行；点击单月行可查看单月明细。
- 区间汇总中期初金额取首月期初，店铺期末余额取末月期末，不再累加期初/期末。
- 明细流水默认按动账时间升序，可切换降序。
- 店铺配置支持维护原始表格字段、汇总表格字段和计算口径说明；导入时自动补充原始字段。

## v1.0.0

- 首个 Qt/PySide6 Windows 桌面版本。
- 支持店铺配置、Excel 流水导入、SQLite 本地落盘。
- 支持按店铺/月度汇总和历史查询。
- 支持期初金额、店铺期末余额录入与结算期末余额核对。
- 支持差异原因说明和差异明细查看。
- 支持手工调整指定汇总列，并记录调整动作。
- 支持流水编辑、删除、月份流水删除和 CSV 导出。
