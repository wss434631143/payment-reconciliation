# v1.0.3

这是 Qt 版财务第三方支付核对程序的正式发布版本。本版本重点解决大数据导入/查询卡顿、CSV 支持、店铺数据隔离、参数配置、筛选分页、备份还原和整体界面体验。

## 主要更新

- 每个店铺使用独立 SQLite 数据库，提升隔离性和查询性能。
- 首页按店铺、报表类型、年月区间查询汇总和明细。
- 导入流水统一在一个弹窗中选择店铺、报表类型、归属年月和文件。
- 支持 Excel 与 CSV 导入，CSV 兼容 UTF-8 BOM 和 GB18030。
- 明细流水支持分页、跳页、表头排序和多字段精确筛选。
- 筛选改为先“加入条件”，再“开始筛选”，减少大表频繁刷新。
- 参数配置可维护冻结栏、活动栏、计算口径和分页行数。
- 汇总表冻结区和活动区之间可拖动调节宽度。
- 差异明细使用表格化页面展示，并保留差异原因和调整说明。
- 导出 XLSX 使用分批读取和流式写入，适合大数据量。
- 导出文件包含汇总表和原始表格，文件名包含店铺、年月区间和时间戳。
- 新增全局设置：开机自启动、点击 X 最小化到托盘。
- 新增备份还原：可备份/还原全部或部分店铺数据库及配置。
- 弹窗按钮改为中文场景文案。
- 明细筛选条件过多时支持自动换行显示。
- 明细流水表格和底部控制/状态区域之间支持拖动调整高度。
- 明细分页按钮与编辑/删除流水按钮合并到一行。
- 导入完成后状态栏显示“导入完成”、新增行数和跳过行数。
- 明细筛选按钮文案调整为“加入添加筛选条件”“清除筛选条件”。
