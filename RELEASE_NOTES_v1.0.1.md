# v1.0.1

本版本重点增强多月份核对、导出和店铺字段配置。

## 新增功能

- 汇总表新增“调整说明”，手工调整后会在汇总中展示调整列、事项、金额和说明。
- 导出汇总表改为 XLSX，包含：
  - `汇总表`
  - `原始表格`
- 月份筛选支持多选，也支持输入区间，例如 `202606-202607`。
- 多月份查询时按店铺显示区间合计，并保留每个月的单月行。
- 点击区间合计行显示区间明细流水，点击单月行显示单月明细流水。
- 明细流水默认按动账时间升序排序，可切换为降序。
- 店铺配置支持维护：
  - 原始表格字段
  - 汇总表格字段
  - 计算口径说明
- 导入店铺表格时，会自动补充该店铺的原始表格字段。

## 核对规则调整

多月份区间汇总时：

- 期初金额取区间首月期初。
- 店铺期末余额取区间末月期末。
- 期初金额和店铺期末余额不累加。
- 明细流水、收入、支出、提现、调整等按区间汇总。
