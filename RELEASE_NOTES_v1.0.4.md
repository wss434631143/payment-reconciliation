# v1.0.4

- 参数配置点击“保存参数”后不再自动关闭窗口，可继续编辑；退出时仍会提示未保存修改。
- 自定义汇总公式支持引用原始表格金额字段，修复自定义字段计算结果为空的问题。
- 自定义汇总公式支持多轮依赖计算，修复“结算期末余额”先于“提现金额”等字段计算时未回头重算的问题。
- 当指定金额字段全为空或 0 时，`其他扣款(不开票)`、`消费者赔付`、`提现金额` 支持按动账场景汇总动账金额取数。
- 备份及还原弹窗新增进度文字和进度条，按准备、处理店铺、完成/失败阶段持续反馈。
- 首页按钮文案调整为“店铺配置”“参数配置”“录入余额”“差异调整”“备份及还原”。
- 修复 Windows“程序和功能”卸载失败的问题，卸载时会关闭正在运行的主程序、删除快捷方式、清理注册表并删除安装目录。
- 主窗口标题栏显示程序图标和版本号；首页大标题保持简洁，不再额外显示图标和版本文本。
- 汇总表冻结窗口默认宽度调整为 300，默认展示“店铺、报表类型、年月”三列，同时不再限制最小宽度，用户可自由拖动。
- 强化安装/卸载流程：安装前会关闭并清理安装目录中的旧版主程序，卸载时会同时关闭当前版本和旧版残留进程，避免旧 exe 占用导致安装目录删不掉。
- 查询汇总、自定义字段取数、动账场景兜底、参数配置字段加载、备份和还原均按分页配置分批处理，并在状态栏或弹窗中持续显示进度，减少大数据场景下的未响应。
- 首页点击“查询”后汇总数据改为后台线程加载，状态栏持续显示汇总、自定义字段和动账场景进度，避免大数据店铺查询时整窗未响应。
