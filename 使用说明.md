# 财务第三方支付核对 Qt 版使用说明

## 启动

双击 `财务第三方支付核对-Qt版.exe`。

## v1.0.1 重点

1. 手工调整后，调整说明会显示在汇总表的 `调整说明` 列。
2. 导出汇总表为 `.xlsx`，包含 `汇总表` 和 `原始表格` 两个 sheet。
3. 月份筛选支持多选，也支持输入区间，例如 `202606-202607`。
4. 多月份汇总时，期初金额取首月期初，店铺期末余额取末月期末，不累加期初/期末。
5. 明细流水默认按动账时间升序，可点击 `动账时间：升序/降序` 切换。
6. 查询区间时，点击区间合计行显示区间明细；点击单月行显示该月明细。
7. 店铺配置中可维护原始表格字段、汇总表格字段和计算口径说明。
8. 导入店铺表格时，系统会自动补充该店铺的原始表格字段。

## 数据位置

数据库保存在 exe 同目录：

`data/reconciliation.sqlite3`

## 计算口径

默认口径：

`收入净额合计 = 订单实付应结 + 平台补贴 + 商家补贴 + 结算运费 + 订单退款`

`支出金额 = 已结算佣金 + 技术服务费`

`结算金额 = 收入净额合计 + 支出金额`

`结算期末余额 = 期初金额 + 收入净额合计 + 支出金额 + 提现金额`

不同店铺可在 `店铺配置 -> 字段/口径配置` 中维护口径说明和字段展示。
