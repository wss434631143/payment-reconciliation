# v1.0.2

本版本修复抖音店铺 Excel 导入兼容问题。

## 修复

- 支持 `动帐时间1`、`动帐流水号` 等“账/帐”字段别名。
- 当表格没有 `动账摘要` 时，自动使用 `动账场景`、`计费类型` 等字段作为摘要/业务类型。
- 导入时不再强制要求固定旧表头，能根据店铺表格字段自动识别并初始化原始字段配置。

## 字段自动映射

- `运费实付` 映射到 `结算运费`
- `平台服务费` 映射到 `技术服务费`
- `实际平台补贴_运费`、`实际平台补贴`、`其他平台补贴` 等汇总到 `平台补贴`
- `服务商佣金`、`渠道分成`、`招商服务费` 等汇总到 `佣金`
